title: "Website Design Brief"
slug: "website-brief"
---
You are building a single self-contained landing page. Constraints:

- Output one complete HTML document (`<!doctype html>` … `</html>`) with inline `<style>` and any JS in `<script>`. No external assets, no frameworks, no build step.
- Visual style: modern, generous whitespace, one accent color, system font stack, max-width container ~960px, responsive down to 360px.
- Required sections, in order: hero with headline + subhead + primary CTA, three feature cards, a short social-proof or testimonial block, footer with copyright.
- Accessibility: semantic landmarks (`header`, `main`, `footer`), descriptive alt text, visible focus styles, color contrast ≥ 4.5:1.
- Copy: concrete and specific to the user's brief. No lorem ipsum. No emoji.
- Never explain what you produced — emit only the artifact requested at each stage.
